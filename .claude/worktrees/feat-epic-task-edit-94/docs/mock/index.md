# モック索引

4 つのビューから入る。

| No  | ビュー                       | 説明                                                                     |
| --- | ---------------------------- | ------------------------------------------------------------------------ |
| 1   | [画面（pages）](pages/)      | 画面ごとに Issue 一覧 → 案リスト → 案の実画面 と辿る                     |
| 2   | [コンポーネント（components）](components/) | 共通コンポーネントの全案を 1 ページに並べたカタログ                      |
| 3   | [スタイル（styles）](styles/) | 画面全体のデザインスタイル。採用 1 案 + 候補案を画面で見比べる           |
| 4   | [Issue 横断（issues）](issues/) | 1 Issue で複数画面を扱う場合に、その Issue から関連画面に最短で辿る      |
