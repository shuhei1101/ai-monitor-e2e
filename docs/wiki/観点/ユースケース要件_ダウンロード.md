# ユースケース要件 観点: ダウンロード

ダウンロード / エクスポート系 UC（CSV / PDF / レポート出力 等）で歩く観点。
`## ユースケース要件` の草案作成時に、以下カテゴリを 1 つずつ走査して該当行を要件として立てる。

## 対象データ

| No | 観点 | 洗い出す内容 |
| --- | --- | --- |
| 1 | 範囲指定 | 全件 / 表示中の絞り込み結果 / 選択したものだけ |
| 2 | フィルター継承 | 一覧のフィルター状態を引き継ぐか |
| 3 | 期間指定 | 日付範囲での絞り込み |
| 4 | 項目選択 | 出力項目を選ばせるか、固定か |

## フォーマット

| No | 観点 | 洗い出す内容 |
| --- | --- | --- |
| 1 | 形式 | CSV / TSV / Excel / PDF / JSON / ZIP |
| 2 | 文字コード | UTF-8 / UTF-8 BOM / Shift_JIS の選択 |
| 3 | 区切り / 改行 | カンマ / タブ、CRLF / LF |
| 4 | ヘッダー行 | ヘッダーの有無、多言語ラベル |
| 5 | 日付フォーマット | ISO 8601 / 「YYYY/MM/DD」/ ローカル形式 |

## 生成方式

| No | 観点 | 洗い出す内容 |
| --- | --- | --- |
| 1 | 同期 / 非同期 | 即時生成ダウンロード / バックグラウンド生成後にメール等で通知 |
| 2 | 進捗表示 | 生成中の進捗バー / 待機表示 |
| 3 | キャンセル | 生成途中のキャンセル可否 |
| 4 | 分割 | 大容量時の分割ダウンロード / ZIP 圧縮 |

## 制限

| No | 観点 | 洗い出す内容 |
| --- | --- | --- |
| 1 | サイズ上限 | 1 回あたりの件数 / ファイルサイズ上限 |
| 2 | 実行頻度 | 短時間の連続ダウンロード制限 |
| 3 | ファイル寿命 | 非同期生成物のダウンロード可能期限 |

## エラー

| No | 観点 | 洗い出す内容 |
| --- | --- | --- |
| 1 | 生成失敗 | 失敗時のメッセージ、リトライ動線 |
| 2 | タイムアウト | 上限超過時の挙動 |
| 3 | 該当なし | 出力対象 0 件時の表示 |

## 履歴

| No | 観点 | 洗い出す内容 |
| --- | --- | --- |
| 1 | ダウンロード履歴 | 過去に生成したファイルの再ダウンロード可否 |
| 2 | 監査ログ | 誰がいつ何をダウンロードしたか記録するか |

## アクセス制御

| No | 観点 | 洗い出す内容 |
| --- | --- | --- |
| 1 | ダウンロード権限 | 誰が実行できるか |
| 2 | データ権限フィルタ | 権限外レコードは自動除外 |
| 3 | 機密度別制限 | PII / 機密データを含む出力の追加制約 |
