# モジュール構成

ai-monitor の構成要素（クラス / 関数 / 関数型 / データモデル）の索引ページ。
詳細は各分類別ファイルへ。

論理名（業務語彙）はドキュメント・口頭で、実名（クラス名 / 関数名 / 型名）はソースコードで使う。
両者の対応は本ページと各分類別ファイルでのみ正規化して管理する。
