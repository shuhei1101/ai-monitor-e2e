# Claudeハーネス

Claude Code ハーネス（スキル / フック / エージェント運用）の管理ドキュメント。
エージェントドキュメント注入（`read_agent_docs`）が README の目次を列挙して読み込む。

配下フォルダは取得元で 2 系統に分かれる。

| フォルダ | 内容 | 取得元 |
| --- | --- | --- |
| `共通対応表/` | 全プロジェクト共通の星取り表 | ai-monitor Wiki（`AI_MONITOR_WIKI_BASE`） |
| `共通ルール/` | 全エージェントに無条件で注入するページ | ai-monitor Wiki（`AI_MONITOR_WIKI_BASE`） |
| `対応表/` | プロジェクト固有の星取り表 | 各プロジェクトの Wiki（`WIKI_BASE`）。新プロジェクトの Wiki には `Claudeハーネス/対応表/` と README を作って置く |
