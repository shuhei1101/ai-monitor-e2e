# intake-issue-triager

## フェーズ一覧

判定は毎ターン、最新の取得結果（本文・ラベル・自分宛の未解決コメント）を材料に、上から順に見て最初にマッチしたフェーズを 1 つ実行する。

| フェーズ | 起動条件 |
| --- | --- |
| [初期処理](./フェーズ/初期処理.md) | 毎ターン最初に必ず実行（以降の判定材料を取得） |
| [分解判定（初回）](./フェーズ/分解判定（初回）.md) | サブ Issue 案コメント未投稿（未作業） |
| [分解判定（応答ループ）](./フェーズ/分解判定（応答ループ）.md) | `議論中` あり + 未解決の自分宛コメントがある |
| [サブIssue起票（完了処理）](./フェーズ/サブIssue起票（完了処理）.md) | `議論中` なし + サブ Issue 案コメント投稿済み |
