# エージェント

エージェントごとの実務ドキュメント（フェーズ索引 + フェーズページ）。
各エージェントの SKILL.md は本フォルダの自分のページを動的注入 / 実行時取得で参照する（書式は `テンプレート/スキル.md`）。
各ページの書式は [テンプレート](./テンプレート/) に従う。
