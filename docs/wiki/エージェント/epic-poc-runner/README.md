# epic-poc-runner

## フェーズ一覧

判定は毎ターン、最新の取得結果（PR 本文・ラベル・自分宛の未解決コメント）を材料に、上から順に見て最初にマッチしたフェーズを 1 つ実行する。

| フェーズ | 起動条件 |
| --- | --- |
| [初期処理](./フェーズ/初期処理.md) | 毎ターン最初に必ず実行（以降の判定材料を取得） |
| [方針固め（初回）](./フェーズ/方針固め（初回）.md) | PoC PR 本文に `## 検証構成` が未記入（本文が `## 紐づく Issue` のみ） |
| [応答ループ](./フェーズ/応答ループ.md) | `議論中` あり + 未解決の自分宛コメントがある |
| [検証実行](./フェーズ/検証実行.md) | `議論中` なし + PoC PR 本文の `## 成功条件` 記入済み + `## 検証結果` 未記入 |
| [完了処理](./フェーズ/完了処理.md) | `議論中` なし + PoC PR 本文の `## 検証結果` 記入済み |
