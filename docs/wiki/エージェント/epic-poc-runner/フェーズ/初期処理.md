# 初期処理

毎ターン（初回・応答ループ・完了処理のどの呼び出しでも）最初に実行する。

## 手順

### 最新状態の取得

MCP `get_issue_or_pr` を呼ぶ:
- `number`: $pr_number
- `is_pr`: true
- `parent`: false
- `sub_issues`: false

続けて MCP `list_addressed_comments` を呼ぶ:
- `number`: $pr_number
- `is_pr`: true
- `addressee`: `epic-poc-runner`
- `include_resolved`: false

### フェーズの決定

フェーズ索引のフェーズ一覧を上から順に見て、起動条件に最初にマッチしたフェーズの手順を実行する。
前ターンの記憶ではなく取得結果を判定材料にする（本文・ラベル・コメントはターン間にユーザーや他エージェントが変更している可能性がある）。
