# ai-monitor テンプレート: PR 本文 / 軽微修正

`layer:chore` が付いた PR の本文書式。

chore PR は **quick-implementer が軽微修正を直接コミットする PR**。
書くのは **紐づく Issue と変更内容の概要** だけ（変更の詳細は変更ファイル一覧で見る）。

## セクション一覧

| セクション | サブセクション | 必須or条件 | 補足 |
| --- | --- | --- | --- |
| `## 紐づく Issue` | - | 必須 | 元の chore Issue 番号 |
| `## 概要` | - | 必須 | 変更内容の説明 |

## `## 紐づく Issue`

### 記述例

```markdown
## 紐づく Issue

- #123 README の typo 修正
```

### 補足

- 元の chore Issue 番号を必須で書く

## `## 概要`

### 記述例

```markdown
## 概要

README のインストール手順の typo（`instal` → `install`）を修正。
```

### 補足

- 何をどう変えたかを 1〜3 行で書く
