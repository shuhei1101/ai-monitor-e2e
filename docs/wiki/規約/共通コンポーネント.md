# ai-monitor: 共通コンポーネント規約

プロジェクト初期に「画面全体で繰り返し使うコンポーネント」を決めるための材料集。
コンポーネント名と「決めるべき観点（What to decide）」だけを並べる。

このページは網羅を狙うが、プロジェクトの要件に合わせて追加・改変してよい。

## 使い方

1. プロジェクト初期で、画面全体で使う共通コンポーネントの方針を決めるタイミングで参照する
2. 一覧から採用するコンポーネントを選ぶ
3. 各コンポーネントの「観点」をプロジェクト側 Wiki に転記し、そのプロジェクトでの決定内容を埋める
4. モック画面の案出しでは、ここで決めたコンポーネントを材料として使う（勝手に独自コンポーネントを足さない）

## コンポーネント一覧

### レイアウト系

| コンポーネント | 決めるべき観点 |
| --- | --- |
| アップシェル | トップバー / サイドバー / フッタの組合せ、フル幅か中央寄せか |
| トップバー | 配置（固定 / 追従）、含める要素（ロゴ / ナビ / 検索 / プロフィール / 通知）、高さ感 |
| サイドバー | 配置（左 / 右）、開閉可否、幅の固定 / 可変、階層対応 |
| ドロワー | 出現方向（左 / 右 / 上 / 下）、モバイル時のサイドバーとの関係 |
| フッタ | 有無、含める要素（コピーライト / リンク / 通知） |
| パンくず | 採用有無、配置位置、最大階層 |

### ナビゲーション系

| コンポーネント | 決めるべき観点 |
| --- | --- |
| タブ | 配置（横 / 縦）、スタイル（下線 / 塗り / ピル）、スクロール対応 |
| ステップインジケータ | 配置（横 / 縦）、戻る可否、現在地表示の強さ |
| ページネーション | 方式（ページャー / 無限スクロール / もっと見る）、ページサイズ選択の可否 |

### 入力系

| コンポーネント | 決めるべき観点 |
| --- | --- |
| テキスト入力 | ラベル位置（上 / 左）、必須マーク、エラー表示位置 |
| テキストエリア | リサイズ可否、最大行、文字数カウンタ |
| セレクト | 単一 / 複数、検索可否、グルーピング |
| チェックボックス | 単独 / グループ、グループ時のレイアウト |
| ラジオ | 配置（横 / 縦）、デフォルト選択の有無 |
| トグル | ラベル位置、即時反映か確定保存か |
| 日付ピッカー | 単日 / 範囲、カレンダー表示、ショートカット（今日 / 今週など）の有無 |
| 時刻ピッカー | 分の粒度、24h / 12h |
| スライダー | 値表示の有無、ステップ、二点指定の可否 |
| ファイルアップロード | ドラッグ&ドロップ、複数選択、プレビュー |

### データ表示系

| コンポーネント | 決めるべき観点 |
| --- | --- |
| テーブル | 行密度、列ピン留め、ソート方式、選択モード、行クリック挙動 |
| カード | サイズの揃え方、影の有無、ヘッダ・フッタの有無 |
| リスト | 区切り線、アバター・アイコン位置、補助情報 |
| バッジ / タグ | 形状（角丸 / 矩形）、色の使い分けルール |
| チップ | 削除可否、選択可否、アイコンの有無 |
| アバター | 形状（円 / 角丸）、サイズ段階、フォールバック |
| チャート | デフォルトの種類（折れ線 / 棒 / 円）、凡例位置、ホバー時の情報 |
| プログレス | バー / 円形、確定 / 不確定、ラベル |

### フィードバック系

| コンポーネント | 決めるべき観点 |
| --- | --- |
| トースト | 出現位置、自動消去時間、種類別の色 |
| モーダル | サイズ段階、背景クリックで閉じるか、エスケープキー挙動 |
| ダイアログ | 用途別の使い分け（確認 / 警告 / 危険操作）、強度別の表現 |
| ボトムシート | モバイル限定か、半開きの可否 |
| ツールチップ | 遅延時間、出現方向の自動調整 |
| スケルトン | 採用有無、形状の作り分け |
| エンプティステート | イラスト / アイコンの有無、CTA の配置 |

### アクション系

| コンポーネント | 決めるべき観点 |
| --- | --- |
| プライマリボタン | 形状（矩形 / ピル）、サイズ段階、ローディング表現 |
| セカンダリボタン | プライマリとの視覚的差（塗り / 枠 / テキストのみ） |
| アイコンボタン | サイズ段階、ツールチップ必須化 |
| FAB | 採用有無、位置、複数アクション展開の方式 |
| メニュー（ケバブ） | アイコン形状、出現方向 |

### 特殊系

| コンポーネント | 決めるべき観点 |
| --- | --- |
| カレンダー | デフォルトビュー、ビュー切替、イベント表示形式 |
| コードブロック | シンタックスハイライト、コピーボタン、行番号 |
| リッチテキストエディタ | 採用有無、ツールバー構成 |
| マップ | 採用有無、ピンの形状、クラスタリング |
